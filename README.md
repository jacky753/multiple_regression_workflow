# multiple_regression_workflow

多重回帰，回帰分析には流れ作業のような決まった手順通りにやるだけの作法がある．その理論，具体的な計算方法，実装についてプログラムとともに記載する．

# 多重回帰モデル 作成の手順 (理論)

## ステップ 1. [データを入手] 
- データを取得

## ステップ 2. [最後に実施するテストデータによる回帰モデルの評価のための準備] 
- データセットについてランダムに選んだ1/5はテストデータとして分けておく．残りの4/5を学習データにする．

## ステップ 3. [学習データの中で特徴量選択] 
- VIF値が10 または 5以上の説明変数について除いた説明変数の組み合わせによって次のステップに進む．

## ステップ 4. [多重回帰モデルの選択] 
- ステップ 3. において複数ある説明変数のパターンからより適切な説明変数の組み合わせを赤池情報量基準(AIC)によって選択する．いわゆる「モデルの選択」と呼ぶ工程である．
- この際，各説明変数の組み合わせにおける多重回帰モデルを評価するために，5-交差検証(Cross Validation) を実施して，AICの平均値を計算することが望ましい．5-CVを実施することで，学習データの偏り，ばらつきによる偶然おきる多重回帰モデルの予測性能の良さ，悪さを回避しやすくなる．「過学習を防ぐ」とも呼ぶ配慮である．

## ステップ 5. [テストデータによる最終的な多重回帰モデルの評価] 
- ステップ 1. にて分けておいたテストデータについて，ステップ 4. で選んだ多重回帰モデルを用いて予測して予測誤差を平均事情誤差(MSE)などで求めて，最終的な多重回帰モデルの予測精度の評価とする．


# 具体的な計算方法

ステップ 4. について，赤池情報量基準(AIC)の計算が最も理解につまずく可能性があるので補足する．

AICは下記の式であらわされる．AICは機械学習，深層学習でいう損失関数(誤差関数) である．この値が小さいほど多重回帰モデルは優れている解釈する．

  AIC = -2 ln(L) + 2k

https://en.wikipedia.org/wiki/Akaike_information_criterion

L : 最大尤度． ln(L) = log_e (L) は尤度関数と呼ばれる．

k : 自由パラメータの数．説明変数の数．

ここで，本質的な意味を考えて，L(最大尤度)は多重回帰モデルを学習して作成した後に決まる決定係数(R^2)を代入して良い．

決定係数は多重回帰モデルがどれほどデータにフィットしているか，説明変数が目的変数をどれほど説明できているかを表しており，

AIC は本来

−2logL

を使います。

しかし線形回帰では

−2logL ∝ log(RSS) ∝ log(1−R^2)

なので、

AIC = nlog(1−R^2) + 2k + 定数

と書けます。

そのため、
「尤度の代わりに R^2を使っているように見える」
のです。

ただし本質的には、
- AIC→ 尤度ベース
- R^2→ RSS を経由して尤度に変換可能

という関係です。

また，決定係数(R^2)は説明変数が増えるほど大きくなりやすい傾向があるので，多重回帰ではadjusted R^2を使う．

https://exploratory.io/note/exploratory/R2-tjU2NHv6
