# 多重回帰モデル 作成の手順 (理論)

## ステップ 1. [データを入手] 
- データを取得

## ステップ 2. [最後に実施するテストデータによる回帰モデルの評価のための準備] 
- データセットについてランダムに選んだ1/5はテストデータとして分けておく．残りの4/5を学習データにする．

## ステップ 3. [学習データの中で特徴量選択] 
- VIF値が10 または 5以上の説明変数について除いた説明変数の組み合わせによって次のステップに進む．

## ステップ 4. [多重回帰モデルの選択] 
- ステップ 3. において複数ある説明変数のパターンからより適切な説明変数の組み合わせを赤池情報量基準(AIC)によって選択する．いわゆる「モデルの選択」と呼ぶ工程である．
- 後述するが，**多重回帰モデルでは AIC = n log(1-R^2) + 2k**を用いる．
- この際，各説明変数の組み合わせにおける多重回帰モデルを評価するために，5-交差検証(Cross Validation) を実施して，AICの平均値を計算することが望ましい．5-CVを実施することで，学習データの偏り，ばらつきによる偶然おきる多重回帰モデルの予測性能の良さ，悪さを回避しやすくなる．「過学習を防ぐ」とも呼ぶ配慮である．

## ステップ 5. [テストデータによる最終的な多重回帰モデルの評価] 
- ステップ 1. にて分けておいたテストデータについて，ステップ 4. で選んだ多重回帰モデルを用いて予測して予測誤差を平均二乗誤差(MSE)などで求めて，最終的な多重回帰モデルの予測精度の評価とする．


# 具体的な計算方法
## 赤池情報量基準(AIC) で使う式と具体的に代入する値について
ステップ 4. について，赤池情報量基準(AIC)の計算が最も理解につまずく可能性があるので補足する．

AICは下記の式であらわされる．AICは機械学習，深層学習でいう損失関数(誤差関数) である．この値が小さいほど多重回帰モデルは優れている解釈する．

  AIC = -2 ln(L) + 2k

https://en.wikipedia.org/wiki/Akaike_information_criterion

L : 最大尤度． ln(L) = log_e (L) は尤度関数と呼ばれる．

k : 自由パラメータの数．説明変数の数．

ここで，幾多の導入を経て，-2 ln(L) は多重回帰モデルを学習して作成した後に決まる決定係数 R^2 を用いてn log(1-R^2) を代わりに用いることができる．

つまり，多重回帰モデルにおいては，

AIC = -2 ln(L) + 2k ∝ n log(1-R^2) + 2k

よって，

> AIC = n log(1-R^2) + 2k

を用いることができる． n は，データ数である．

以下は導入についてである．途中式を省いた概略的な話である．

この式は，多重回帰モデルにおける尤度関数L を，多重回帰モデルの予測誤差 ε = model y_i - data y_i が正規分布に従うと仮定して， 各データの多重回帰モデルとの誤差が独立であることを仮定すると，全体の尤度 L は各確率密度の積になることから

L(β,σ2) = ∏_(i=1...n) ​f(yi​)

⇒ 

L = ∏_(i=1...n) 1/root(2πσ^2) exp( −(model y_i​ − data y_​i​)^2/2σ^2​ )

⇒

指数関数の積は指数の和になるので，

L = (2πσ^2) ^(-n/2) exp( -1/2σ^2 Σ_(i=1...n) (model y_i - data y_i)^2 )

と記述することができる．

こちらの L を次の RSS の定義 と 両辺 対数を取って整理して，

RSS = ∑_(i=1...n) ​(model y_i​ − data y_​i​)^2 より

log(L) = −n/2 ​( log(2π) + 1 + log(RSS​/n) )

⇒

−2log(L) = nlog(RSS/n​) + 定数

という導出からL と 残差平方和(RSS) との関係が求まる．

R^2 = 1 − RSS/TSS

TSS は 全平方和 であり，

log(RSS) = log(1−R^2) + 定数​

⇒

log(RSS/n) ∝ log(1−R^2) + 定数​

⇒

−2log(L) ∝ nlog(1-R^2​) + 定数

かなり長い導出と言える．

決定係数は多重回帰モデルがどれほどデータにフィットしているか，説明変数が目的変数をどれほど説明できているかを表しており，多重回帰における尤度 L は，「この回帰モデルが現在のデータをどれくらいうまく説明できるか」を確率的に表したものである．

尤度関数(または単に尤度と呼ぶ)は，下記で表す．

ここで，データ X(確率変数 X) ，パラメータ θ(確率密度関数 f(X|θ) を定義する分散 σ, 平均値 μなど)と置くと，データ X が観察されたとき，パラメータ θ の尤もらしさを

L(θ|X) = f(X|θ) 

と表す．

https://en.wikipedia.org/wiki/Likelihood_function

https://ja.wikipedia.org/wiki/%E5%B0%A4%E5%BA%A6%E9%96%A2%E6%95%B0

線形回帰では，

-2ln(L) ∝ −2log(L) ∝ log(RSS) ∝ log(1−R^2)

であるので，

AIC = nlog(1−R^2) + 2k + 定数

と書ける．多重回帰モデル同士で 定数 は変わらないため比較するときに 必要ない．ただし，比例∝ の式を比較するときに，定数部分は省いても構わないが，掛け算している係数部分は比較に影響するので省くことはできない．掛け算しているということは比例関係にあるということだからである．

よって，

AIC = n log(1−R^2) + 2k

を用いる．

そのため，
「尤度 L の代わりに 1-R^2を使っているように見える」
ということである．

ただし本質的には，
- AIC→ 尤度ベース
- R^2→ RSS を経由して尤度に変換可能

という関係でである．

## テストデータによる評価段階ではadjusted R^2 調整済みR^2を利用すること
決定係数(R^2)は説明変数が増えるほど大きくなりやすい傾向があるので，多重回帰ではadjusted R^2を使う．
赤池情報量基準(AIC)では，導出の流れとしてadusted R^2は利用すべきではなく，通常の決定係数R^2を使うことに気を付ける．
